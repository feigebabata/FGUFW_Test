Rogue技能系统:
    Rogue类型游戏即像<Rogue>的游戏
    设定:
        严重的死亡惩罚机制 (一条命)
        随机获得能力 (技能池加权随机)
        关卡与敌人随机 (避免重复游玩的疲劳)

仿<20 Minutes Till Dawn>:
    肉鸽+弹幕

    玩家/怪/树           0
